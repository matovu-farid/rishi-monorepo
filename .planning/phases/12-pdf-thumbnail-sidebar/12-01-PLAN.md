Component signature: `export function ThumbnailSidebar({ onClose, onNavigate }: { onClose: () => void; onNavigate: (pageNumber: number) => void })`

Implementation:
1. Read atoms: `numPages = useAtomValue(pageCountAtom)`, `currentPage = useAtomValue(pageNumberAtom)`, `pdfProxy = useAtomValue(pdfDocumentProxyAtom)`
2. Create `containerRef = useRef<HTMLDivElement>(null)`
3. Create virtualizer:
   ```typescript
   const thumbVirtualizer = useVirtualizer({
     count: numPages,
     getScrollElement: () => containerRef.current,
     estimateSize: () => THUMBNAIL_HEIGHT + GAP,
     overscan: 3,
   });
   ```
4. On mount, scroll to current page in thumbnail list:
   ```typescript
   useEffect(() => {
     if (currentPage > 0) {
       thumbVirtualizer.scrollToIndex(currentPage - 1, { align: 'center' });
     }
   }, []);
   ```
5. Handle click: `const handleClick = (pageNum: number) => { onNavigate(pageNum); onClose(); };`
6. Render: A scrollable `div` with `ref={containerRef}` and `className="overflow-y-auto h-full p-2"`. Inside, a positioned container div with `style={{ height: thumbVirtualizer.getTotalSize(), position: 'relative' }}`. Map over `thumbVirtualizer.getVirtualItems()` rendering each as an absolutely positioned div (same pattern as main virtualizer in pdf.tsx):
   ```tsx
   {thumbVirtualizer.getVirtualItems().map((item) => {
     const pageNum = item.index + 1;
     return (
       <div
         key={item.key}
         style={{
           position: 'absolute',
           top: 0,
           left: 0,
           width: '100%',
           height: `${item.size}px`,
           transform: `translateY(${item.start}px)`,
         }}
         className="flex flex-col items-center"
       >
         <Thumbnail
           pageNumber={pageNum}
           width={THUMBNAIL_WIDTH}
           pdf={pdfProxy ?? undefined}
           onItemClick={() => handleClick(pageNum)}
           className={cn(
             'cursor-pointer border-2 rounded transition-colors',
             currentPage === pageNum
               ? 'border-blue-500 shadow-md'
               : 'border-transparent hover:border-gray-300'
           )}
         />
         <span className="text-xs text-gray-500 mt-1">{pageNum}</span>
       </div>
     );
   })}
   ```

Note: Pass `pdf={pdfProxy ?? undefined}` to Thumbnail so it uses the already-loaded PDFDocumentProxy instead of requiring a wrapping `<Document>`. This avoids double-loading the PDF.

**Step 4: Create thumbnail navigation handler**

Add after `onItemClick` function. CRITICAL: Reset `bookNavigationStateAtom` to `Idle` before calling `setPageNumber`. Without this, if the state is `Navigating` (e.g., from a prior scroll), `setPageNumberAtom` is a no-op and the thumbnail click silently fails (see RESEARCH.md Pitfall 3).

```typescript
function onThumbnailNavigate(pageNumber: number) {
  // Reset navigation state to Idle so setPageNumber is not a no-op
  // (setPageNumberAtom skips if BookNavigationState is Navigating)
  setBookNavState(BookNavigationState.Idle);
  virtualizer.scrollToIndex(pageNumber - 1, {
    align: "start",
    behavior: "smooth",
  });
  setPageNumber(pageNumber);
  updateBookLocationMutation.mutate({
    bookId: book.id.toString(),
    location: pageNumber.toString(),
  });
}
```
