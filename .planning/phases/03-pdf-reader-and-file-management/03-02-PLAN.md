Changes:

a) **Unified import flow:** Replace the single `handleImport` with a format chooser. When user taps the import button (FAB or empty state CTA), show an Alert with three buttons:
```typescript
Alert.alert('Import Book', 'Choose file format', [
  { text: 'EPUB', onPress: () => doImport('epub') },
  { text: 'PDF', onPress: () => doImport('pdf') },
  { text: 'Cancel', style: 'cancel' },
])
```
The `doImport` function calls either `importEpubFile()` or `importPdfFile()` based on the choice, then calls `loadBooks()` on success.

b) **Format-aware routing:** Update `handleBookPress` to route based on format:
```typescript
const handleBookPress = useCallback((book: Book) => {
  if (book.format === 'pdf') {
    router.push(`/reader/pdf/${book.id}`)
  } else {
    router.push(`/reader/${book.id}`)
  }
}, [router])
```

c) **Delete functionality:** Add `handleDelete` function:
```typescript
const handleDelete = useCallback((book: Book) => {
  Alert.alert(
    'Delete Book',
    `Are you sure you want to delete "${book.title}"? This cannot be undone.`,
    [
      { text: 'Cancel', style: 'cancel' },
      {
        text: 'Delete',
        style: 'destructive',
        onPress: () => {
          // Delete from DB
          deleteBook(book.id)
          // Delete book files from disk
          const bookDir = new Directory(new Directory(Paths.document, 'books'), book.id)
          if (bookDir.exists) {
            bookDir.delete()
          }
          // Reload library
          loadBooks()
        },
      },
    ]
  )
}, [loadBooks])
```

d) Pass `onDelete={handleDelete}` to each `BookRow` in the FlatList renderItem.

e) Update the import button label: change the FAB accessibilityLabel to "Import Book" (already correct).

f) Reload books when screen comes into focus (so after returning from reader, the list reflects any changes). Add `useFocusEffect` from expo-router:

```typescript
import { useRouter, useFocusEffect } from 'expo-router'
// ...
useFocusEffect(useCallback(() => { loadBooks() }, [loadBooks]))
```
Remove the existing useEffect that calls loadBooks on mount since useFocusEffect covers that case.
