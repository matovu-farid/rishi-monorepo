1. Create `apps/mobile/lib/sync/engine.ts`:
   - Import `db` from `@/lib/db`
   - Import `books, syncMeta` from `@rishi/shared/schema`
   - Import `eq` from `drizzle-orm`
   - Import `apiClient` from `@/lib/api`
   - Import `PushRequest, PushResponse, PullResponse` from `@rishi/shared/sync-types`

   Add a module-level `let isSyncing = false` mutex.

   Export `async function sync(): Promise<void>`:
   - If isSyncing, return immediately
   - Set isSyncing = true, wrap in try/finally to reset
   - Call push() then pull()
   - Catch all errors with console.warn (never throw to caller)

   Implement `async function push()`:
   - Query `db.select().from(books).where(eq(books.isDirty, true)).all()`
   - If no dirty records, return
   - POST to `/api/sync/push` with `{ changes: { books: dirtyBooks } }` via apiClient
   - Parse PushResponse
   - For each conflict, update local record with server values and isDirty: false
   - For each successfully pushed book, set isDirty: false and syncVersion: data.syncVersion

   Implement `async function pull()`:
   - Get lastSyncVersion from syncMeta table (default 0)
   - GET `/api/sync/pull?since_version={lastVersion}` via apiClient
   - Parse PullResponse
   - For each remote book: check if local record exists
     - If exists and NOT locally dirty: update with remote values, isDirty: false
     - If exists and IS locally dirty: skip (local changes take precedence until pushed)
     - If not exists: insert with isDirty: false, filePath: '' (file not downloaded yet), coverPath: null
   - Update syncMeta with new syncVersion and lastSyncAt: Date.now()

2. Create `apps/mobile/lib/sync/triggers.ts`:
   - Import `AppState` and `AppStateStatus` from `react-native`
   - Import `sync` from `./engine`

   Module-level variables: syncInterval, writeDebounce, appStateSubscription (all initially null)

   Export `function startSyncTriggers()`:
   - Add AppState change listener: if state === 'active', call sync()
   - Start setInterval at 5 * 60 * 1000 ms calling sync()
   - Call sync() immediately for initial sync

   Export `function triggerSyncOnWrite()`:
   - Clear existing writeDebounce timeout if any
   - Set new timeout at 2000ms calling sync()

   Export `function stopSyncTriggers()`:
   - Clear interval, clear debounce timeout, remove AppState listener

3. Wire sync into the app:
   - Read the tabs layout or main tab screen to find the right place
   - Add a useEffect in the authenticated tabs area (check `apps/mobile/app/(tabs)/_layout.tsx` or `apps/mobile/app/(tabs)/index.tsx`)
   - In the useEffect: call `startSyncTriggers()`, return `stopSyncTriggers` as cleanup
   - This should only run once when the user is authenticated
